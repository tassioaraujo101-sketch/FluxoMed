# FluxoMed

Primeira versão funcional do FluxoMed com foco em atendimento educacional estruturado por fluxogramas clínicos.

## O que já está funcional
- Interface em estilo chat clínico
- Sidebar com especialidades
- Busca de fluxogramas por texto
- Lista de resultados por especialidade
- Sugestões rápidas clicáveis
- Fluxo completo de suspeita de AVC
- Módulos: anamnese, exame físico, conduta inicial, medicações, o que não fazer e referências
- Campos editáveis para documentação defensiva

## Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS

## Como rodar
```bash
npm install
npm run dev
```
